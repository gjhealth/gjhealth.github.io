##### 扫一扫

name：

```
QRCodeJSBridge
```

params:

```
{
    "title":"请扫描二维码/条形码",
    "light":true,
    "codeType":0 //0 条形码 ,1 二维码
}
```

callback:

```
{ @"QRCodeValueString" : codeString}
```
